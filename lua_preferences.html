PlayerPrefs

Table of Contents

  • PlayerPrefs example

The Unity PlayerPrefs system stores and accesses player preferences between game sessions.

PlayerPrefs example

Here's an example of using PlayerPrefs from Lua.

1 -- Saving a value to preferences
2 playerprefs.SetInt("SaveName", 1)
3 playerprefs.Save()
4 
5 -- Using a value from preferences
6 local v = playerprefs.GetInt("SaveName")
7 print(v) -- Will print out 1

FungusPrefs

The FungusPrefs class is a wrapper around PlayerPrefs that adds support for save slots.

Basically, if you want to store simple values use PlayerPrefs. If you want to store values using multiple player profiles, you should use FungusPrefs. The slot variable is an integer [0..] and key is a string.

1 -- Deletes all saved values for all slots.
2 prefs.DeleteAll()
3 
4 -- Removes key and its value from this save slot.
5 prefs.DeleteKey(slot, key)
6 
7 -- Returns the float value associated with this key in this save slot, it it exists.
8 prefs.GetFloat(slot, key, defaultValue)
9 
10 -- Returns the int value associated with this key in this save slot, it it exists.
11 prefs.GetInt(slot, key, defaultValue)
12 
13 -- Returns the string value associated with this key in this save slot, it it exists.
14 prefs.GetString(slot, key, defaultValue)
15 
16 -- Returns true if the key exists in this save slot.
17 prefs.HasKey(slot, key)
18 
19 -- Writes all modified prefences to disk.
20 prefs.Save()
21 
22 -- Sets the value of the preference identified by key for this save slot.
23 prefs.SetFloat(slot, key, value)
24 
25 -- Sets the value of the preference identified by key for this save slot.
26 prefs.SetInt(slot, key, value)
27 
28 -- Sets the value of the preference identified by key for this save slot.
29 prefs.SetString(slot, key, value)
30 
31 -- Returns the combined key used to identify a key within a save slot.
32 prefs.GetSlotKey(slot, key)
