97 # Map # {#Map}
98 Sets the outValue mapping of a value that currently exists between a set of numbers to another set of numbers.
99 E.g. a value of 5 between 0 and 10, mapped to 0-20 would result in 10.
100 
101 Does not clamp between ranges, use a Fungus.Clamp before or after this command for that if is desired.
