4 # What is Fungus? # {#faq_what_is_fungus}
5 %Fungus is a free open source tool for creating storytelling games, with a strong focus on accessibility for novice game developers. Users all over the world are now using it to create a wonderfully diverse range of storytelling games.
6 
7 # Who makes Fungus? # {#faq_who_makes}
8 %Fungus is developed by [Snozbot], a small independent games company. Chris Gregan is the main developer of %Fungus.
9 
10 # Where do I get Fungus? # {#faq_where_get}
11 %Fungus is available on the [Unity Asset Store] and in the download section of this website.
12 
13 # Is Fungus really free to use? # {#faq_really_free}
14 Yes. %Fungus has an Open Source M.I.T. license. You can use %Fungus with any version of the Unity game engine, including the freely available personal edition.
15 
16 # Can I sell the apps I make with Fungus? # {#faq_can_sell}
17 Yes. We do not charge a license fee.
18 
19 # What platforms does Fungus support? # {#faq_platforms}
20 %Fungus runs on any platform supported by the Unity game engine. See here for a list of supported platforms.
21 
22 # Do I need to be a programmer to use Fungus? # {#faq_non_coders}
23 No. %Fungus can be used to make a variety of story-driven games with no coding required.
24 
25 # Is Fungus useful for programmers? # {#faq_coders}
26 Yes. %Fungus provides an easy way to add visual novel features to any Unity game. It is also possible to extend %Fungus to customise the workflow for your team. Checkout the documentation and tutorials for more information.
27 
28 # Where can I learn more about Fungus? # {#faq_learn_more}
29 We provide extensive tutorials and documentation on our website here. There is also a comprehensive online training course suitable for beginners at Udemy.
30 
31 # I've made something cool with Fungus. Wanna see? # {#faq_show_and_tell}
32 Most definitely! Please tweet [gofungus] and/or post an item in the announcements section of our forum.

34 # How can I upload my Fungus game for others to see? # {#faq_where_upload}
35 After creating a build, you can use a hosting service such as [itch.io], [gamejolt.com] or [newgrounds.com]
36 
37 # How can I report a bug or suggest new features for Fungus? # {#faq_bug_reports}
38 Please use the forum to report bugs and request new features. You can also checkout our github issues list to see what we have planned.
39 
40 # How long will I have to wait for a bug-fix? # {#faq_how_long}
41 If a bug is critical and we can't offer a sensible workaround, we will try to fix it as soon as possible. Otherwise, we will aim to fix the issue in the next release.
42 
43 # Can I get my hands on beta versions of Fungus? # {#faq_beta_version}
44 The very latest version is always available at our [github repository]

46 # How can I contribute to Fungus? # {#faq_contribute}
47 There are lots of ways to help the %Fungus community to grow!
48 
49 1. Answer questions and post tips on the forum
50 2. Contribute code either via a Github pull request or post code snippets in the forums.
51 3. Improve the documentation by editing the Docs folder on github and send in a pull request.
52 4. Let us know on the forum if you've made a game with %Fungus so we can add it to the site.
53 5. If you write an article, blog post, podcast, etc. about %Fungus send it in and we'll add it to the @ref community_tutorials section.
54 
55 # Are you planning to integrate Fungus into other game engines? # {#faq_other_engines}
56 Not currently. We're sticking with Unity for the foreseeable future.
