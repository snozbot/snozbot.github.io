Detailed Description

A simple struct wrapping a reference to a Fungus Block.

Allows for BlockReferenceDrawer. This is the recommended way to directly reference a fungus block in external c# scripts, as it will give you an inspector field that gives a drop down of all the blocks on a flowchart, in a similar way to what you would expect from selecting a block on a command.
