345 ======
346 
347 Major release with powerful new Lua scripting support and many small improvements and fixes.
348 This release should be backwards compatible with projects created using Fungus 2. If you have any upgrading issues let us know on the forum.
349

359 ## Added
360 - FungusLua: Lua scripting support for Fungus via wrapper components for using MoonSharp in Unity. #281, #317, #334, #237, #235, #232, #224
361  - LuaEnvironment component: Execution environment for running Lua scripts.
362  - LuaUtils component: Extends LuaEnvironment with lots of useful features.
363  - LuaBindings: Maps Unity objects & components to Lua variables for use in Lua scripts.
364  - LuaScript: Runs Lua code from a text file or from a string property.
365  - LuaStore: Stores variables in a global table which persists across scene loads.
366  - FungusModule: A set of utility functions for scripting Unity and Fungus from Lua.
367  - FungusPrefs: An improved version of PlayerPrefs that can be easily used from Lua.
368  - ExecuteHandler: Listens for any standard Unity event and calls a method on a component in the gameobject. #247
369  - ExecuteLua command: Run some Lua script in a Fungus command. Return values can be stored in Fungus variables.
370  - PODTypeFactory: Utility factory class for instantiating Plain-Old-Data (POD) types like Color, Vector3, etc.
371  - Lots of FungusLua example scenes
372  - Fungus documentation now has an extensive section on LuaScripting.
373 - StringDataMulti: Like StringData, but uses a multi-line textbox in the inspector.
374 - StopBlock command: Stop executing the named block.
375 - Improved string substitution system. Now works with Lua global variables and Lua string table, as well as Flowchart variables.
376 - Extend the string substitution system yourself using the new ISubstitutionHandler interface.
377 - Added TaskManager library to Thirdparty folder. Allows better control over coroutine execution.
378 - Show Line Numbers option in Flowchart. Shows the command index in the inspector (off by default). #231
379 - Play Animation State command. Plays an animation state directly without a transition. #378
380 - Open URL command #382
381 - Links to community articles in the help docs #385
382 - InfoText.cs component for displaying help information in the top-left of screen
383 - "Play from Selected" and "Stop All and Play" context menu options in Block command list
384 - Added Command Index property to Call command
385 - LuaStore example scene to demonstrate persisting Lua variables between scene loads
386 - Use stage.show(), stage.showPortrait & stage.hide() to control stage & portraits from Lua #490
387  See FungusExamples/FungusLua/Narrative/PortraitController.unity example scene
388 - Portrait functionality moved to new PortraitController utility class for easier scripting.
389 - Say and Menu Dialogs now support standard input manager (joystick / controller support) #507 #210
390 - Menu options can now be picked with keyboard only
391 - Fast forward using Shift in Say Dialogs is now done using the Cancel input (Escape key by default).
