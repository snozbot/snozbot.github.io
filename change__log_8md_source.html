216 This release should be backwards compatible with projects created using Fungus 2. If you have any upgrading issues let us know on the forum.

227 ## Added
228 - FungusLua: Lua scripting support for Fungus via wrapper components for using MoonSharp in Unity. #281, #317, #334, #237, #235, #232, #224
229  - LuaEnvironment component: Execution environment for running Lua scripts.
230  - LuaUtils component: Extends LuaEnvironment with lots of useful features.
231  - LuaBindings: Maps Unity objects & components to Lua variables for use in Lua scripts.
232  - LuaScript: Runs Lua code from a text file or from a string property.
233  - LuaStore: Stores variables in a global table which persists across scene loads.
234  - FungusModule: A set of utility functions for scripting Unity and Fungus from Lua.
235  - FungusPrefs: An improved version of PlayerPrefs that can be easily used from Lua.
236  - ExecuteHandler: Listens for any standard Unity event and calls a method on a component in the gameobject. #247
237  - ExecuteLua command: Run some Lua script in a Fungus command. Return values can be stored in Fungus variables.
238  - PODTypeFactory: Utility factory class for instantiating Plain-Old-Data (POD) types like Color, Vector3, etc.
239  - Lots of FungusLua example scenes
240  - Fungus documentation now has an extensive section on LuaScripting.
241 - StringDataMulti: Like StringData, but uses a multi-line textbox in the inspector.
242 - StopBlock command: Stop executing the named block.
243 - Improved string substitution system. Now works with Lua global variables and Lua string table, as well as Flowchart variables.
244 - Extend the string substitution system yourself using the new ISubstitutionHandler interface.
245 - Added TaskManager library to Thirdparty folder. Allows better control over coroutine execution.
246 - Show Line Numbers option in Flowchart. Shows the command index in the inspector (off by default). #231
247 - Play Animation State command. Plays an animation state directly without a transition. #378
248 - Open URL command #382
249 - Links to community articles in the help docs #385
250 - InfoText.cs component for displaying help information in the top-left of screen
251 - "Play from Selected" and "Stop All and Play" context menu options in Block command list
252 - Added Command Index property to Call command
253 - LuaStore example scene to demonstrate persisting Lua variables between scene loads
254 - Use stage.show(), stage.showPortrait & stage.hide() to control stage & portraits from Lua #490
255  See FungusExamples/FungusLua/Narrative/PortraitController.unity example scene
256 - Portrait functionality moved to new PortraitController utility class for easier scripting.
257 - Say and Menu Dialogs now support standard input manager (joystick / controller support) #507 #210
258 - Menu options can now be picked with keyboard only
259 - Fast forward using Shift in Say Dialogs is now done using the Cancel input (Escape key by default).
